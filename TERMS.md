## 2. Description of the App

TranslateME is a Canva Apps SDK plugin that exports the current design page as a PNG image, sends it to Anthropic's Claude AI API for translation analysis, and writes selected translations back into your Canva design. It supports 26 target languages and provides 3 translation variants per text element for user review.
